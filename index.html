Welcome to Access Code 3.X - 2016 JavaScript Workshop

What is this thing?

Welcome to the Access Code 2015 bootcamp! In this workshop we'll be teaching you the basics of JavaScript and some of the fundamentals of programming. During this class you'll learn about expressions, variables, Arrays, Loops, Conditionals, and Functions, and use these concepts to write your own programs!

Game Plan

This workshop also serves as part of the Access Code application process. After this workshop, you'll be given an assessment containing questions about reading and writing JavaScript. The assessment will test your understanding of what you will learn during the workshop, as well as your ability and initiative to teach yourself new concepts in a short period of time. We'll look at the assessment as part of your application.

Environment

We will all be using the Google Chrome browser. Make sure you are on the latest version.

We will be using CodePen to write and run small JavaScript programs. Please close the HTML and CSS boxes by clicking on the 'X' in the top right of each box. Click on the 'console' button in the bottom left to expand the console. In the JavaScript tab we will put our code, and in the Console tab press Cmd+Shift+5 to execute your code and see the output, and press the clear button to clear the console.

I recommend downloading Sublime Text and copying your code out of CodePen into Sublime text to keep track of all the code you've written over the course of this workshop. We will occassionally refer back to code from earlier sections later on in the course, so having that code handy will be very valuable. I personally like to create a /user/code folder (next to your Documents folder for example) and save all my code in there.

Resources

My favorite JavaScript resources are at w3schools and specifically the JavaScript Tutorial for explanations about the language and the JavaScript Reference for a comprehensive guide to the fundamental tools, libraries, and built-in objects and types of the language.
