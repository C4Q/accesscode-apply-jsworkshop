Lecture: Variables

What is a Variable?

A good analogy is a labeled box. Outside is the label, inside is something, and it could be anything!

(Stolen from Khan Academy)

Officially: A variable is a container. It has both a name and a value. You declare a new variable with the var keyword, and assign it (aka give it) a value with the = sign. 

var variableOne = 1
var variableTwo = "Two"
var bensFavLanguage = "JavaScript"
console.log("Ben's favorite language is: " + bensFavLanguage)

In JavaScript the value of a variable can change, but the name cannot. However, you can assign variables with values from other variables. 

var bensFavLanguage = "JavaScript"
var studentsFavLanguage = "French"
console.log("Student's favorite language was " + studentsFavLanguage)
// It was French!

// .... after this lecture ....
studentsFavLanguage = bensFavLanguage
console.log("Student's favorite language was " + studentsFavLanguage)
// Now it's JavaScript!

As usual, the w3schools page on variables is a fantastic resource worth reading and visiting often. Here are some other fun facts and features of variables:

  • Variables can be used and reused as many times as you want
  • Variables can hold different kinds or types of data (Numbers, Strings, and more!)
  • The name of a variable is very important for you and others reading your code.
  • The name of a variable can contain letters, numbers, underscores, and dollar signs. However it cannot start with a number.

Evaluation / Execution of code

The word we use to describe turning a set of code into a single value is called evaluate or execute. This is useful to think about in situations where we assign the result of some math to a variable: 

var x = 10
var y = 20
var someVariable = (100 * x / y) - x*y
x = 0
y = 0
console.log(someVariable)

The value of some variable is decided at the moment of assignment when that line of code gets executed and the expression on the right side of the assignment gets evaluated.

Useful and important syntax

var someNumber = 0

//Increment: take a value and increase it by 1
someNumber = someNumber + 1
someNumber += 1 // This and the above line can be any number
someNumber++
++someNumber

//Decrement: take a value and decrease it by 1
someNumber = someNumber - 1
someNumber -= 1 // This and the above line can be any number
someNumber--
--someNumber

//Multiply: take a value and multiply it by x
someNumber = someNumber * x
someNumber *= x

//Divide: take a value and divide it by x
someNumber = someNumber / x
someNumber /= x
