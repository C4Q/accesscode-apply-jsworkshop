Lecture: Variables

What is a Variable?

A good analogy is a labeled box. Outside is the label, inside is something, and it could be anything!

(Stolen from Khan Academy)

Officially: A variable is a container. It has both a name and a value. You declare a new variable with the var keyword, and assign it (aka give it) a value with the = sign. 

var variableOne = 1
var variableTwo = "Two"
var bensFavLanguage = "JavaScript"
console.log("Ben's favorite language is: " + bensFavLanguage)

In JavaScript the value of a variable can change, but the name cannot. However, you can assign variables with values from other variables. 

var bensFavLanguage = "JavaScript"
var studentsFavLanguage = "French"
console.log("Student's favorite language was " + studentsFavLanguage)
// It was French!

// .... after this lecture ....
studentsFavLanguage = bensFavLanguage
console.log("Student's favorite language was " + studentsFavLanguage)
// Now it's JavaScript!

Other fun facts and features of variables:

  • Variables can be used and reused as many times as you want
  • Variables can hold different kinds or types of data (Integers, Strings, and more!)
  • The name of a variable is very important for you and others reading your code.
  • The name of a variable can contain letters, numbers, underscores, or dashes. However it cannot START with a number or a dash (but it can start with an underscore).
