Use console.log(myVariable) to print out each variable to the console. You can also prepend it with a string and use the + operator to squish them together (the fancy name for that is to concatenate or join them together). 

console.log("The value of [something] is " + someVariable)

NOTE: If the value of the variable is NOT a String and you try and concatenate it with a string, JavaScript will do a nice thing and convert it to a String for you. However this can be dangerous, JavaScript is not very smart and might convert it incorrectly. This might not be an issue for simple examples, but consider yourself warned!

Math (Expressions)

You can do math on variables too as long as they are numbers! Let's find out your birth year by subtracting your age (let's use the variable you defined in the last step) from the current year: 

var currentYear = 2016
var myBirthYear = currentYear - myAge
console.log("My birth year is " + myBirthYear)

It can get quite a bit more complicated: 

var companyRevenue = 1000000
var officeRent = 250000
var employeeSalary = 75000
var numberOfEmployees = 4
var taxRate = 0.4
var profit = (companyRevenue - officeRent - employeeSalary * numberOfEmployees) * taxRate
console.log("Profit this year was " + profit)

Try changing the values like the rent, or the number of employees, and see how profit changes. This is why programming is so powerful!

Useful and important syntax

var someNumber = 0

//Increment: take a value and increase it by 1
someNumber = someNumber + 1
someNumber += 1 // This and the above line can be any number
someNumber++
++someNumber

//Decrement: take a value and decrease it by 1
someNumber = someNumber - 1
someNumber -= 1 // This and the above line can be any number
someNumber--
--someNumber

//Multiply: take a value and multiply it by x
someNumber = someNumber * x
someNumber *= x

//Divide: take a value and divide it by x
someNumber = someNumber / x
someNumber /= x

The word we use to describe turning a set of code into a single value is called evaluate or execute. This is useful to think about in situations where we assign the result of some math to a variable: 

var x = 10
var y = 20
var someVariable = (100 * x / y) - x*y
x = 0
y = 0
console.log(someVariable)

The value of some variable is decided at the moment of assignment when that line of code gets executed and the expression on the right side of the assignment gets evaluated.
